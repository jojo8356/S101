# S101

University course website (S101) with presentation and session/schedule pages.

## Pages

- **Presentation** — course overview (`index.html`)
- **Sessions** — schedule and session details (`seances.html`)

## Tech Stack

- HTML5, CSS3

## Usage

```bash
open html/index.html
# or
python3 -m http.server 8000
```
