Implémentation d'un besoin client

Objectifs

  • Créer un site web complet en HTML5 et CSS pur, sans JavaScript
  • Rendre l’interface responsive, utilisable sur PC et smartphone
  • Séparer clairement le contenu (HTML) et la mise en forme (CSS)
  • Produire un code propre, valide et commenté (HTML + CSS valides W3C)

Context

Suite à une panne informatique du système de base de données de la grande surface «E.Carrechan», l’équipe informatique ne dispose plus que de fichiers journaliers de tickets de caisse. Vous devez aider les équipes différentes équipes du magasin à gérer au mieux les comptes, stocks, clients etc. en attendant que la base de données soit rétablie.

Fonctionnalités

Lecture complète du CSV

Affichage propre et complet du fichier CSV par la technique de lecture par colonne

Logiciel prêt à l'emploi

Dès que l'on télécharge le code, il suffit juste de taper 'make all'. Les options peuvent etre afficher par l'option -h, --help: affichage de la documentation, On peut tester le code avec ./tests.sh
